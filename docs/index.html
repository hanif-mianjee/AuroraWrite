How It Works

Simple, Powerful, Private

Get started in seconds and write with confidence anywhere on the web

1

Install the Extension

Add AuroraWrite to Chrome from the Web Store or load it as an unpacked extension for development.

2

Configure Your AI Provider

Choose from 6+ AI providers including Groq (free), OpenAI, Gemini, Claude, and more. Enter your API key.

3

Start Writing

Type in any text field on the web. AuroraWrite automatically analyzes your text and shows suggestions.

4

Review & Accept

Click on underlined issues to see suggestions. Accept with one click or select text to transform it.

Under the Hood

Built for Performance

Innovative block-based analysis minimizes API calls and maximizes efficiency

Block-Based Analysis

Text is split into logical blocks. Only changed blocks are re-analyzed, reducing API calls by up to 80%.

Stability Pass System

Verification passes ensure accuracy with confidence scoring. Blocks stabilize at 80% confidence.

Hash-Based Detection

Cryptographic hashing detects dirty blocks instantly. No redundant processing of unchanged content.

Shadow DOM Isolation

All UI components use Shadow DOM to prevent style conflicts with host pages.

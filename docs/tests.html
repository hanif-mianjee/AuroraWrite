Developer Test Page

Use this page to test AuroraWrite functionality locally. Each section contains text fields with sample errors.

Getting Started

  1. Build the extension: npm run dev or npm run build
  2. Load the dist/ folder as an unpacked extension in Chrome
  3. Configure your API key in the extension options
  4. Click on any text field below to start testing
  5. Type or edit text to trigger analysis (700ms debounce)
  6. Hover over underlined issues to see suggestions (300ms delay)
